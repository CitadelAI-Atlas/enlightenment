# 00 - Project Charter

*Written 2026-06-11*

## Mission

Design the best long-form meditation music program we can, grounded in published
theory, generated by code we control completely, and tested as honestly as our
methods allow, then publish all of it for the world to use, verify, and extend.

The deeper aim is understanding: how does a mind absorb an extended period of
structured sound, and which structural choices actually matter? We pursue that aim
cautiously. The history of this genre is full of confident claims with thin
evidence; our contribution is to separate what holds up from what does not, in
public.

## Why medium-length sessions

Receptive music-therapy programs (GIM) and breathwork music protocols run far longer
than a typical meditation track, because a complete arc of settling, deepening,
opening, and return needs room. We adopt that medium-length range and treat exact
duration as an experimental variable: the arc is specified as a normalized contour
(percentages of total duration) and rendered at multiple candidate lengths for
comparison.

## Deliverables

1. **The program**: a gapless render in five movements at each candidate duration,
   plus per-movement stems so experimental variants can be swapped without
   regenerating everything.
2. **The generator**: an open-source, fully parameterized synthesis pipeline. Every
   theoretical claim is a knob.
3. **The experiment**: variants differing in exactly one variable; validated
   self-report instruments for absorption and meditative depth (for example the
   Tellegen Absorption Scale and the Toronto Mindfulness Scale) plus a session log;
   optional heart-rate-variability data.
4. **The publication**: this repository plus an interactive web player where
   listeners toggle experimental layers and submit session reports, making the
   experiment crowd-sourced and ongoing.

## Experimental variables (initial set)

| Variable | Conditions | Blindable? |
|---|---|---|
| Session duration | candidate medium lengths, compared head to head | no |
| Entrainment layer | on / off | yes |
| Entrainment method | binaural / isochronic | yes |
| Breath-pacing layer | on / off | partially |
| Quiet interval at peak | present / absent | no |
| Reference tuning | 440 Hz / 432 Hz | yes |

## Epistemic ground rules

1. Claims are graded (evidence-backed / speculative / aesthetic) everywhere they
   appear.
2. Hypotheses are committed publicly before trials; null and negative results get
   equal prominence.
3. Small-sample, self-report trials are treated as hypothesis-generating, never as
   proof.
4. We prefer measurable proxies (heart-rate variability, blind discrimination) over
   impressions wherever possible.
5. No medical or therapeutic claims; no guarantee that any sound produces any state.
