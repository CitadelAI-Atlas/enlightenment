# Lab Journal - 2026-06-11 - Session 01: Foundations

## What happened

Project conceived and chartered: an open, published experiment in long-form
meditation music, aimed at understanding how the mind absorbs extended structured
sound.

## Decisions made

1. **Medium-length program.** Receptive music-therapy (GIM) and breathwork music
   protocols run far longer than typical meditation tracks because a complete arc
   needs room. We adopt that range and make exact duration an experimental
   variable, specified as a normalized contour and rendered at candidate lengths.
2. **Five-movement arc** (Threshold / Ascent / Peak / Heart / Return), peak near
   55-60 percent of session duration. Drafted as `specs/arc-v0.md`.
3. **Gapless single program**, not discrete tracks. Movements are the experimental
   unit, rendered as swappable stems.
4. **Everything parameterized**: session duration, entrainment on or off and by
   which method, breath layer, quiet interval at peak, reference tuning. All are
   experiment variables; several are blindable.
5. **Honesty policy**: evidence-backed, speculative, and purely aesthetic elements
   are labeled as such (432 Hz and Solfeggio carried as test variables only, with
   no claims).
6. **Creative commitments** logged in `docs/03-creative-direction.md`: natural 1/f
   variation, breath as conductor, harmonic-series melody, a leitmotif memory arc,
   spatial width as a journey parameter, detuning that re-converges to unison,
   polyrhythmic blur, singular events, and the player itself as the experiment
   instrument.

## Hypotheses on record (pre-trial, informal; formalized per trial later)

- H1: The breath-pacing layer will produce measurable heart-rate-variability shifts
  versus the no-breath variant.
- H2: Entrainment-on variants will be subjectively distinguishable from and
  preferred over entrainment-off, with a smaller effect than H1.
- H3: 440 Hz versus 432 Hz reference tuning will be indistinguishable under blind
  conditions.

## Same-day addenda

1. **Course correction on mission framing.** Direction from the project lead: be
   cautious and as scientific as reasonably possible about the core aim,
   understanding how the mind absorbs sound. A "Scientific stance" section was
   added to the README and "Epistemic ground rules" to the charter: graded claims,
   hypotheses before trials, small-sample honesty, measurable proxies over
   impressions, no medical claims.
2. **Published.** Repository at github.com/CitadelAI-Atlas/enlightenment and docs
   site at enlightenment-sigma.vercel.app, so corrections like the one above are
   part of the public record.
3. **Publication style adopted** (`docs/04-publication-style.md`): meditation
   framing, neutral citation language, normalized durations instead of fixed
   lengths, sober tone, and typographic rules. All founding documents were revised
   to comply, and the public history was re-issued the same day so the record
   begins in compliance.

4. **Generator v1 built** while the literature research pass runs. Spec encoded as
   machine-readable JSON (`specs/arc-v0.json`); renderer streams 24-bit 48 kHz
   stereo WAV in constant memory; all experimental variables are command-line
   flags. Verified: byte-identical renders from identical seeds, distinct output
   across variants, and a measured interaural difference of exactly 5.00 Hz at the
   session midpoint against a spec value of 5.0. Design and known limitations in
   `docs/05-generator-design.md`.
